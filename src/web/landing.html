Convert File

Convert a DXF file to Gcode. Many programs can output DXF files.
Try going to File >> Save as... >> File Type: DXF in AutoCAD
or go to the File >> Export menu in Adobe Illustrator

The software does not support splines, so make sure to convert all of those to 'arcs'
